About These Maps

These interactive maps were generated using a custom map renderer that creates web-viewable versions of Minecraft worlds. You can navigate around, zoom in/out, and see all the builds that were created during each server's lifetime.

Each map preserves the legacy of everything created during each server's lifetime. If you were part of these servers, see if you can find your builds!

Maps are periodically archived when servers are reset or closed. Check back regularly for new additions.
